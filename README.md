# Zettabyte Life Test Section 1

## Setup
Before run the app:

```
$ npm install
$ cp .env.example .env
$ npm run seed
```

## Running
To run the app,

```
$ npm start
```

## Test
To run the test,

```
$ npm run test
```

## Usage
Go to `http://localhost:3000/graphql` to use the playground, or hit the URL
from Postman or VSCode Rest Client with the `{"query": "<query>"}` in the
request body.
